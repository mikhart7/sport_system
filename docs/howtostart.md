#Как запустить программу

Для запуска программы нужно передать 4 или 5 аргументов, в зависимости от режима работы.

Для генерации заявочных списков нужно передать 4 аргумента
##Создание стартовых протоколов
Для создания стартовых протоколов необходимо ввести 4 аргумента
* Путь к файлу со списками мероприятий
* Путь к файлу со списками групп, в которых будут соревноваться участники
* Путь к файлу с описанием каждой дистанции
* Путь к папке, в которой находятся заявочные списки команд

все пути должны быть указаны полностью например:
>C:\Users\admin\courses.csv
>
все данные должны быть занесены в таблицу, поэтому файлы должны иметь расширение ".csv"
>##При создании стартовых протоколов, все участники с неккоректными данными снимаются с соревнования!


##Обработка результатов
Для создания файла с общими результатами необходимо указать 5 аргументов программы:
* Путь к файлу со списками мероприятий
* Путь к файлу со списками групп, в которых будут соревноваться участники
* Путь к файлу с описанием каждой дистанции
* Путь к папке, в которой находятся заявочные списки команд
* Путь к файлу с записями прохождения контрольных точек всеми участниками

все пути должны быть указаны полностью например:
>C:\Users\admin\splits.csv
>
все данные должны быть занесены в таблицу, поэтому файлы должны иметь расширение ".csv"
